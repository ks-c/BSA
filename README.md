# BSA (BrainStorm with AI) 🧠🤖

BSA 是一个基于纯前端技术实现的多 AI 协作头脑风暴系统。在这个平台中，你可以配置多个拥有独立人格、独立 API 和独立提示词的 AI 大模型，让它们在一个聊天界面中相互协作、探讨甚至互相启发。

本项目无需任何后端服务器，直接通过浏览器调用兼容 OpenAI 格式的 API 即可运行，轻量、便捷且高度可定制。

## ✨ 核心特性

- 🧑‍🤝‍🧑 **多 AI 协作**：支持设置多个 AI 角色，每个角色拥有独立的 API Key、模型名称和 System Prompt。
- 🧠 **三种工作模式**：
  - **同步思考模式**：所有激活的 AI 同时处理用户问题，快速发散思维。
  - **轮流接力模式**：AI 依次进行回答，后一个 AI 基于前一个 AI 的输出继续推演，实现深度链式思考。
  - **指定 @ 回答模式**：用户可定向 `@` 某个 AI，AI 在回复时也能 `@` 其他 AI 触发连锁反应（内置防死循环机制）。
- 💾 **本地数据记忆**：所有的 API 密钥、模型配置和提示词修改均实时保存在浏览器的 `localStorage` 中，刷新不丢失，保证数据隐私安全。
- 🛠️ **预设协作团队**：内置 4 个专属角色（项目总负责人、实验设计、理论整理、文献搜索），开箱即用。
- ⚡ **纯原生实现**：无需 Node.js，无需安装依赖，基于原生 HTML5 + CSS3 + ES6 JavaScript 开发。

## 🛠️ 技术栈

- **前端 UI**：HTML5, CSS3 (Flexbox 响应式布局，现代化极简设计)
- **逻辑交互**：原生 JavaScript (ES6+ 模块化)
- **数据存储**：Browser `localStorage`
- **网络通信**：Fetch API (兼容 OpenAI 规范的接口)

## 🚀 快速上手

1. **获取项目**
   `git clone https://github.com/ks-c/BSA.git`
2. **运行项目**
   无需构建，直接在浏览器中双击打开根目录下的 `index.html` 文件即可。
3. **配置 API**
   - 进入任意一种工作模式，在左侧的“AI 控制面板”中，点击展开各个 AI 角色的设置。
   - 填入你使用的 API 接口地址（必须是兼容 OpenAI 格式的地址，如各大代理站或大模型厂商提供的兼容接口）。
   - 填入对应的 `API Key` 和 `模型名称`。
   - 配置完成后，系统会自动将其保存到本地。
4. **开始头脑风暴**
   - 勾选你需要激活的 AI 角色。
   - 在右侧输入框发送你的问题或指令，观察不同 AI 的思维碰撞！

## 🤖 预设角色说明

系统默认预设了一个科研/项目攻坚团队，用于演示 AI 之间如何通过互相 `@` 实现任务闭环：

- **项目总负责人**：负责统筹全局，把控进度，汇总意见并给出最终方案。可通过 `@` 呼叫其他专家。
- **实验设计**：将理论转化为可操作的实验方案。需要背景资料时可 `@理论整理` 或 `@文献搜索`。
- **理论整理**：构建核心理论框架。可 `@文献搜索` 获取信息，或 `@实验设计` 进行验证。
- **文献搜索**：提供可靠的背景资料和学术动态，为团队提供知识库支持。

*(💡 提示：你可以在设置面板中自由修改这些角色的名称和系统提示词，以适应软件开发、小说创作、剧本杀等其他场景。)*

## ⚠️ 注意事项

- **跨域问题 (CORS)**：纯前端直接调用某些官方大模型 API 可能会遇到浏览器的跨域限制。建议使用支持跨域的第三方接口，或者在本地配合浏览器跨域解除插件使用。
- **API 费用控制**：在使用 `@` 指定模式时，AI 之间的互相 `@` 可能会触发多轮自动对话。请留意您的 API Token 消耗，遇到无限探讨的情况可随时点击底部的“停止生成”按钮强行中断。
